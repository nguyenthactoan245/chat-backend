🔄 LUỒNG HOẠT ĐỘNG CHI TIẾT

1. ĐĂNG KÝ / ĐĂNG NHẬP
──────────────────────
   Browser ──POST /auth/register──► AuthController
                                         │
                                    bcrypt.hash(password)
                                         │
                                    INSERT INTO users
                                         │
   Browser ◄──{ token, username, userId }─┘

2. KẾT NỐI REALTIME
────────────────────
   Browser ──WebSocket connect──► ChatGateway
   Browser ──emit("userOnline")──► Gateway lưu vào Map
   Gateway ──broadcast("onlineUsers")──► TẤT CẢ clients

3. GỬI TIN NHẮN
────────────────
   Browser A ──emit("sendPrivateMessage")──► ChatGateway
                                                 │
                                           saveMessage() → DB
                                                 │
              ┌──────────────────────────────────┤
              ▼                                  ▼
         Browser A                          Browser B
    ◄──"newPrivateMessage"──────────────"newPrivateMessage"──►

4. CHAT VỚI BOT AI
───────────────────
   Browser ──emit("sendBotMessage")──► ChatGateway
                                            │
                                       BotService.getReply()
                                            │
                                       Groq API (internet)
                                       Model: Llama 3.3 70B
                                            │
   Browser ◄──emit("newBotMessage")─────────┘
🛠️ CÔNG NGHỆ SỬ DỤNG
Layer	Công nghệ	Mục đích
Frontend	Angular 21	UI, routing, component
Realtime	Socket.IO (client)	Gửi/nhận tin nhắn realtime
Backend	NestJS (Node.js)	REST API + WebSocket server
Realtime	Socket.IO (server)	WebSocket gateway
Auth	JWT + bcrypt	Đăng nhập, bảo mật
ORM	TypeORM	Kết nối & query database
Database	PostgreSQL	Lưu users, messages
AI Bot	Groq API (Llama 3.3)	Trả lời tự động
Deploy	Docker + Docker Compose	Container hóa backend + DB
